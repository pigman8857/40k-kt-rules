# Kill Team Tac Ops Reference

Tac Ops are secondary objectives that players attempt to complete during a battle to earn victory points. Each kill team belongs to one or more **archetypes** (Seek & Destroy, Security, Infiltration, or Recon); if a kill team has more than one archetype, the player selects which one to use for that battle. Each player builds a six-card deck from their archetype's Tac Ops (plus up to three faction-specific Tac Ops if their faction has them), then draws and selects three cards to take into the battle. The selected cards are **kept secret** — opponents do not know which Tac Ops you have chosen.

Each Tac Op has two scoring conditions (marked with bullet points). You can score VP from each condition once only, and you cannot score VP from any condition before the Tac Op is **revealed**. Each card specifies exactly when it must be revealed.

---

## Seek & Destroy

Kill teams with this archetype focus on incapacitating enemy operatives, often rewarding aggression, precision kills, and dominating the mid-board firefight.

| Tac Op | Reveal When | How to Score VP |
|---|---|---|
| **Headhunter** | An enemy LEADER operative is incapacitated | • 1VP when an enemy LEADER is incapacitated • 1VP if this happens in the first or second Turning Point |
| **Challenge** | Target Reveal step of the first Turning Point | • Select one enemy and one friendly operative; 1VP if that enemy is incapacitated by that friendly operative • 1VP if they were within engagement range of each other when the kill was made |
| **Rout** | Target Reveal step of any Turning Point | • 1VP when an enemy operative is incapacitated wholly within the opponent's drop zone • 1VP if this condition is met again in a subsequent Turning Point |
| **Execution** | End of any Turning Point in which more enemy operatives than friendly operatives were incapacitated | • 1VP if this condition was met in two or more Turning Points • 1VP if this condition was met in three or more Turning Points |
| **Deadly Marksman** | Secretly select one friendly operative as your Marksman; reveal when the Marksman incapacitates an enemy while wholly within your own drop zone via a Shoot action | • 1VP when the Marksman incapacitates a further enemy operative • 1VP if the Marksman survives the battle (and the first condition has been scored) |
| **Rob and Ransack** | A friendly operative incapacitates an enemy within engagement range while more than engagement range from all other enemy operatives | • 1VP when this condition is first met • 1VP if that friendly operative survives until the end of the battle |

## Security

Kill teams with this archetype focus on holding territory, protecting their own forces, and denying the opponent access to key areas of the killzone.

| Tac Op | Reveal When | How to Score VP |
|---|---|---|
| **Seize Ground** | Target Reveal step of the first Turning Point; secretly select a terrain feature with the Heavy trait more than engagement range from your drop zone | • 1VP if the total APL of friendly operatives within engagement range of the selected terrain exceeds the total APL of enemy operatives within engagement range • 1VP if there are no enemy operatives within engagement range and the total APL of friendly operatives within engagement range is 4 or more |
| **Hold the Line** | Target Reveal step of any Turning Point after the first | • 1VP if there are no enemy operatives wholly within your drop zone at the end of the Turning Point • 1VP if this condition is met again in a subsequent Turning Point |
| **Protect Assets** | Target Reveal step of any Turning Point | • 1VP if two or more enemy operatives are incapacitated within engagement range of objective markers during this Turning Point • 1VP if this condition is met again in a subsequent Turning Point |
| **Damage Limitation** | Target Reveal step of any Turning Point after the first | • 1VP if no friendly operatives are incapacitated during this Turning Point • 1VP if this condition is met again in a subsequent Turning Point |
| **Plant Banner** | Secretly select one friendly operative carrying the Banner token; reveal when that Banner token is dropped | • 1VP if the Banner token ends up within engagement range of the opponent's drop zone • 2VP instead if it is wholly within the opponent's drop zone (note: this card includes a special DESTROY BANNER mission action, 1AP, for the opponent) |
| **Central Control** | Target Reveal step of any Turning Point | • 1VP if the total APL of friendly operatives within engagement range of the killzone centre exceeds that of enemy operatives • 1VP if this condition is met again in a subsequent Turning Point |

## Infiltration

Kill teams with this archetype focus on penetrating deep into enemy territory, disrupting their operations, and completing covert mission actions.

| Tac Op | Reveal When | How to Score VP |
|---|---|---|
| **Capture Hostage & Infiltrate** | A friendly operative incapacitates an enemy while within engagement range and more than engagement range from all other enemy operatives | • 1VP when that friendly operative is removed from the killzone (escaping with the hostage) • 1VP if this occurs during the third or fourth Turning Point |
| **Behind Enemy Lines** | Target Reveal step of any Turning Point | • 1VP if one or more friendly operatives are wholly within the opponent's drop zone and more than engagement range from all enemy operatives • 1VP if this condition is met again in a subsequent Turning Point |
| **Upload Viral Code** | Target Reveal step of any Turning Point | • 1VP when a friendly operative performs the Upload Viral Code mission action (2AP) • 1VP if this is performed again in a subsequent Turning Point |
| **Implant** | A friendly operative strikes an enemy operative during a Fight action | • 1VP when a friendly operative implants an enemy (choosing to implant instead of inflicting damage) • 1VP if two further enemy operatives are implanted over the course of the battle |
| **Sabotage** | Target Reveal step of any Turning Point | • 1VP when a friendly operative performs the Sabotage mission action (2AP) on a terrain feature • 1VP when a friendly operative Sabotages a second terrain feature |
| **Interloper** | Secretly select one friendly operative as your Interloper; reveal when the Interloper performs the Interlope mission action (1AP, once per battle) | • Remove the Interloper from the killzone and score 2VP (single combined condition worth 2VP) |

---

## Recon

Kill teams with this archetype focus on gathering information, establishing positions across the killzone, and denying the opponent tactical awareness.

| Tac Op | Reveal When | How to Score VP |
|---|---|---|
| **Mark Target** | A friendly operative performs a Shoot action | • 1VP when a friendly operative performs the Mark Target action (choosing to mark the enemy LEADER instead of shooting, inflicting no damage) • 1VP if this is performed again in a subsequent Turning Point |
| **Triangulate** | Target Reveal step of any Turning Point | • 1VP if friendly operatives have performed the Triangulate mission action (1AP) from two different killzone edges • 1VP additionally if Triangulate has been performed from three different killzone edges |
| **Plant Signal Beacon** | A friendly operative performs the Plant Signal Beacon mission action (2AP, once per battle) | • 2VP if the Signal Beacon token is within the opponent's drop zone at the end of the battle (single combined condition worth 2VP) |
| **Vantage** | Target Reveal step of any Turning Point | • 1VP when a friendly operative performs the Vantage mission action (2AP) to secure a Vantage Point • 1VP if a friendly operative performs this again in a subsequent Turning Point |
| **Retrieval** | Target Reveal step of the first Turning Point; the opponent places a Retrieval token | • 1VP if a friendly operative is controlling the Retrieval token at the end of the battle • 1VP additionally if that operative is carrying the token |
| **Overrun** | Target Reveal step of any Turning Point | • 1VP if a friendly operative is wholly within each quarter of the killzone and more than engagement range from the centre • 1VP if this condition is met again in a subsequent Turning Point |

---

> **Selection summary:** In matched play, each player builds a six-card Tac Ops deck (up to three faction-specific cards, the rest from their chosen archetype), then draws two cards at a time and picks one — repeated until three cards are selected. You do not reveal which three you have chosen. Each card specifies exactly when it is revealed during play; you cannot score VP from a condition before its Tac Op is revealed. Each of the two conditions on a card can be scored once only, giving a maximum of 2VP per card and 6VP total from Tac Ops.
